# Contributing to TeraDef

Thank you for considering contributing to TeraDef! We value all contributions, whether they are through code, documentation, or discussions. Please follow the guidelines outlined here to ensure a smooth contribution process.

## How to Contribute
1. Fork the repository.
2. Create a feature branch.
3. Write your code.
4. Write tests for your code.
5. Submit a pull request.

## Code Style
Ensure your code adheres to our coding standards. Check the `CODE_STYLE.md` file for detailed guidelines.
